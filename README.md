# 🚢 Permit Declaration System (TradeWeb™ Live)

A full-stack web application designed to solve real-world logistics workflow problems, specifically focusing on customs permit declarations and cargo management.

## 🚀 Live Demo
**Access the live application here:** [https://permit-system-8j2g.onrender.com/](https://permit-system-8j2g.onrender.com/)

---

## 🛠️ The Problem & Solution
In the logistics industry, handling customs permits requires precision and structured data management. This project replaces manual tracking with a digital workflow that allows for:
* **Permit Submission**: Digital forms for cargo and transport details.
* **Real-time Storage**: Instant data persistence for audit trails.
* **Job Management**: Unique job number tracking for every declaration.

## 💻 Tech Stack
* **Frontend**: HTML5, CSS3 (Flexbox/Animations), JavaScript (DOM Manipulation & Fetch API).
* **Backend**: Java 21 with Spring Boot (REST API, JPA/Hibernate).
* **Database**: Supabase (PostgreSQL).
* **DevOps**: Dockerized deployment on Render.

## ⚙️ Key Features
- [x] **Full CRUD Operations**: Create, Read, Update, and Delete permit records.
- [x] **Database Integration**: Secure connection to Supabase via HikariPool.
- [x] **Responsive UI**: Clean, industry-standard design for desktop and mobile use.
- [x] **Production Ready**: Automated CI/CD pipeline using GitHub and Render.
